# 3.13 Actuación 13: Modernización y Ampliación de la Red WiFi Municipal

## 3.13.1 Objeto del manual de usuario

La presente sección describe el uso funcional de la red WiFi municipal modernizada por parte
de personal técnico-operativo responsable de monitorización, gestión de acceso y atención de
incidencias de conectividad.

## 3.13.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Operador de red | Supervisión de estado de puntos de acceso |
| Técnico municipal TIC | Configuración básica y resolución de incidencias |
| Responsable de servicio | Seguimiento de calidad y continuidad del servicio |

## 3.13.3 Acceso al panel de gestión WiFi

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al panel de administración de la red WiFi municipal. |
| 2 | Iniciar sesión con usuario autorizado. |
| 3 | Seleccionar área geográfica o conjunto de puntos de acceso. |
| 4 | Verificar estado de conectividad y carga de servicio. |

## 3.13.4 Operación diaria

| Acción | Instrucción |
| --- | --- |
| Revisar estado de AP | Consultar disponibilidad y eventos recientes por equipo. |
| Monitorizar calidad de servicio | Comprobar parámetros básicos de conectividad y uso. |
| Identificar zonas con baja cobertura | Detectar puntos críticos y registrar necesidad de actuación. |
| Consultar incidencias abiertas | Priorizar resolución según impacto en usuario final. |

## 3.13.5 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| No hay conexión en una zona | Verificar estado del AP y enlace de red asociado. |
| Conexión inestable | Revisar saturación, interferencias y eventos del equipo. |
| Portal de acceso no disponible | Comprobar estado del servicio de autenticación/cautivo. |
| Reincidencia en el mismo punto | Escalar para revisión técnica de infraestructura. |

## 3.13.6 Buenas prácticas

1. Revisar diariamente eventos de red y alarmas críticas.
2. Mantener inventario actualizado de puntos de acceso y zonas de cobertura.
3. Documentar tiempos de resolución e incidencias repetitivas.
4. Coordinar actuaciones en campo con planificación municipal y proveedores.
