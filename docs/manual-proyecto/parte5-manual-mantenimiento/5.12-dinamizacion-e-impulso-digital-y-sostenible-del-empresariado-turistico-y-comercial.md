# 5.12 Actuación 12: Dinamización e Impulso Digital y Sostenible del Empresariado Turístico y Comercial - Mantenimiento

## 5.12.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo de
la plataforma de dinamización digital, incluyendo auditoría de datos de empresas, validación
de integridad de contenidos y gestión de incidencias operacionales.

## 5.12.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Base de datos de empresas | Integridad, backup, limpieza | Diaria |
| Servidor de aplicación | Rendimiento, monitorización | Diaria |
| Sistema de campañas | Auditoría de estado, archivado | Semanal |
| Almacenamiento de contenido | Validación de integridad de archivos | Mensual |
| Comunicaciones con empresas | Validación de notificaciones | Semanal |

## 5.12.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación de integridad | Verificar que datos de empresas son consistentes | Diaria | DBA |
| Auditoría de campañas | Revisar estado de campañas activas y archivadas | Semanal | Responsable campañas |
| Limpieza de datos obsoletos | Archivar información de empresas inactivas | Mensual | Administrador |
| Test de comunicaciones | Validar que notificaciones se envían correctamente | Semanal | Equipo técnico |
| Backup de base de datos | Realizar copia de seguridad completa | Diaria | Automatizado |

## 5.12.4 Procedimiento de validación de integridad

| Paso | Instrucción |
| --- | --- |
| 1 | Ejecutar consulta de integridad de base de datos. |
| 2 | Validar que no hay registros huérfanos de empresas. |
| 3 | Verificar que campañas están vinculadas correctamente. |
| 4 | Auditar datos de contacto de empresas activas. |
| 5 | Investigar inconsistencias identificadas. |
| 6 | Aplicar correcciones si es necesario. |
| 7 | Documentar resultado de validación. |

## 5.12.5 Auditoría de campañas

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Revisión de campañas activas | Auditar que campaña está dentro de fechas | Semanal |
| Análisis de participación | Revisar número de empresas adheridas | Semanal |
| Validación de criterios | Verificar que empresas participantes cumplen requisitos | Mensual |
| Auditoría de historial | Revisar campañas archivadas y resultados | Trimestral |

## 5.12.6 Limpieza de datos obsoletos

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Identificar empresas inactivas | Revisar que no han participado en >6 meses | Mensual |
| Archivar datos | Mover registros de empresas inactivas a histórico | Trimestral |
| Limpieza de campaña | Eliminar datos de campañas cerradas y archivadas | Trimestral |
| Validación post-limpieza | Verificar que datos retenidos son consistentes | Post-limpieza |

## 5.12.7 Gestión de incidencias operacionales

| Tipo de incidencia | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Empresa no puede acceder | Revisar estado de cuenta y permisos | Reactivar cuenta si es necesario |
| Datos no se actualizan | Auditar permisos de edición | Restaurar permisos o revisar validaciones |
| Notificación no se envía | Revisar configuración de notificaciones | Testear canal y reenviar si aplica |
| Campaña se cierra incorrectamente | Revisar lógica de cierre automático | Restaurar o reabrir si es necesario |

## 5.12.8 Procedimiento de recuperación ante corrupción

| Paso | Instrucción |
| --- | --- |
| 1 | Detectar corrupción mediante validación de integridad. |
| 2 | Aislar sistema y notificar a administrador. |
| 3 | Restaurar desde backup más reciente. |
| 4 | Validar integridad de datos restaurados. |
| 5 | Reconciliar cambios post-backup si es posible. |
| 6 | Reanudar operación normal. |
| 7 | Investigar causa de corrupción. |

## 5.12.9 Documentación de mantenimiento

| Documento | Contenido | Actualización |
| --- | --- | --- |
| Log de auditoría | Campañas revisadas, participantes, resultado | Semanal |
| Registro de incidencias | Problemas identificados y resoluciones | Continua |
| Matriz de participación | Empresas activas por campaña | Semanal |
| Informe de resultados | Efectividad y ROI de campañas | Trimestral |

## 5.12.10 Buenas prácticas de mantenimiento

1. Ejecutar auditoría semanal de campañas y participación de empresas.
2. Mantener validación de integridad automatizada con alertas.
3. Documentar todas las decisiones operacionales en registro de auditoría.
4. Realizar backup diario con pruebas de restauración mensuales.
5. Coordinar mantenimiento con responsables de dinamización.
