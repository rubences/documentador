# 3.10 Actuación 10: Sistema de Control Inteligente del Tráfico

## 3.10.1 Objeto del manual de usuario

La presente sección describe el uso funcional del sistema inteligente de gestión del tráfico
para personal municipal encargado de supervisión semafórica, respuesta ante incidencias y
ajuste operativo de parámetros de movilidad.

## 3.10.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Operador de tráfico | Seguimiento en tiempo real del estado semafórico |
| Responsable de movilidad | Ajuste de parámetros y evaluación de rendimiento |
| Técnico de operaciones | Gestión de incidencias y validación de comunicaciones |

## 3.10.3 Acceso al sistema de control

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la plataforma de control semafórico desde el puesto autorizado. |
| 2 | Iniciar sesión con credenciales de operación. |
| 3 | Seleccionar zona o intersecciones de supervisión. |
| 4 | Verificar recepción de telemetría y estado de controladores. |

## 3.10.4 Operación diaria

| Acción | Instrucción |
| --- | --- |
| Monitorizar intersecciones | Consultar estado de ciclos, fases y tiempos. |
| Revisar alertas de servicio | Identificar fallos de comunicación o equipos en degradado. |
| Aplicar ajustes autorizados | Modificar parámetros operativos según procedimiento municipal. |
| Validar efecto del cambio | Comprobar impacto en flujo y tiempos de retención. |

## 3.10.5 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| Una intersección no comunica | Revisar estado del controlador y enlace de comunicaciones. |
| Tiempos anómalos de ciclo | Verificar configuración activa y restaurar plantilla validada. |
| Alerta persistente no resuelta | Registrar caso y escalar al soporte técnico especializado. |
| Desfase entre estado real y panel | Refrescar telemetría y comprobar sincronización del sistema. |

## 3.10.6 Buenas prácticas

1. Aplicar cambios en franjas de menor impacto cuando sea posible.
2. Registrar toda modificación operativa para trazabilidad.
3. Validar en campo los ajustes relevantes en puntos críticos.
4. Coordinar actuaciones con Policía Local y servicios de movilidad.
