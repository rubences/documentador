# 5.9 Actuación 9: Sistema de Información de Aparcamientos Públicos - Mantenimiento

## 5.9.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo del
sistema de información de aparcamientos públicos, incluyendo calibración de sensores de conteo,
validación de integridad de datos y gestión de fallos de equipamiento.

## 5.9.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Sensores de presencia | Calibración, limpieza de zona | Trimestral |
| Sensores magnéticos | Validación de detección, calibración | Trimestral |
| Contadores de tráfico | Validación de precision | Semanal |
| Sistema central de control | Monitorización de rendimiento | Diaria |
| Base de datos de ocupación | Integridad, backup, auditoría | Diaria |

## 5.9.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación de ocupación | Verificar coherencia de conteos en tiempo real | Semanal | Equipo técnico |
| Test de precisión | Comparar conteos con validación manual | Mensual | Técnico en campo |
| Limpieza de zonas de sensor | Remover obstáculos que afecten detección | Trimestral | Equipo mantenimiento |
| Inspección física de sensores | Revisar estado y posición de detectores | Trimestral | Técnico en campo |
| Auditoría de integridad | Verificar que base de datos es consistente | Mensual | DBA |

## 5.9.4 Procedimiento de validación de ocupación

| Paso | Instrucción |
| --- | --- |
| 1 | Seleccionar aparcamiento para validación. |
| 2 | Realizar conteo manual completo de plazas ocupadas. |
| 3 | Registrar dato del sistema de ocupación. |
| 4 | Calcular diferencia entre manual y sistema. |
| 5 | Si diferencia > 5%, investigar causa. |
| 6 | Aplicar corrección manual si es necesario. |
| 7 | Documentar resultado y acciones realizadas. |

## 5.9.5 Mantenimiento de sensores de conteo

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Inspección visual | Revisar sensor no está obstruido | Mensual |
| Limpieza de óptica | Remover polvo de lentes de sensor | Trimestral |
| Test de funcionamiento | Validar que sensor detecta movimiento | Mensual |
| Calibración de distancia | Ajustar parámetros si sensibilidad se degrada | Trimestral |

## 5.9.6 Procedimiento de recalibración de sensor

| Paso | Instrucción |
| --- | --- |
| 1 | Parar tráfico en zona de sensor si es posible. |
| 2 | Ejecutar secuencia de test del sensor. |
| 3 | Validar que detecta presencia de vehículo. |
| 4 | Ajustar distancia de detección si es necesario. |
| 5 | Validar que no hay falsas alarmas. |
| 6 | Registrar calibración realizada. |
| 7 | Reanudar operación normal. |

## 5.9.7 Gestión de fallos de conteo

| Tipo de falla | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Conteo anómalamente alto | Validar en campo y revisar sensor | Recalibrar sensor o revisar si hay problemas |
| Conteo anómalamente bajo | Inspeccionar si sensor obstruido | Limpiar sensor y recalibrar |
| Conteo se congela | Verificar conectividad de nodo | Reiniciar nodo o revisar comunicación |
| Discrepancia entrada/salida | Revisar si hay sensor dañado | Recalibrar sensores o reemplazar |

## 5.9.8 Auditoría de integridad de datos

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Validación de consistencia | Verificar que entradas + salidas ≈ ocupación actual | Diaria |
| Análisis de anomalías | Identificar cambios abruptos no realistas | Semanal |
| Validación cruzada | Comparar datos de diferentes sensores de mismo aparcamiento | Semanal |
| Recuperación de datos | Restaurar datos incompletos desde backup si es necesario | As-needed |

## 5.9.9 Mantenimiento de infraestructura

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Inspección de cables | Revisar estado de conexiones eléctricas y de datos | Trimestral |
| Limpieza de zona | Remover obstáculos que afecten detección | Mensual |
| Validación de drenaje | Comprobar que agua no afecta sensores | Trimestral |
| Test de redundancia | Validar que datos se respaldan correctamente | Mensual |

## 5.9.10 Documentación de mantenimiento

| Documento | Contenido | Actualización |
| --- | --- | --- |
| Log de validaciones | Fecha, aparcamiento, resultado, responsable | Cada validación |
| Registro de fallos | Problemas identificados y resoluciones | Continua |
| Matriz de precisión | Precisión por aparcamiento y sensor | Mensual |
| Plan de reemplazo | Sensores próximos a fin de vida | Trimestral |

## 5.9.11 Buenas prácticas de mantenimiento

1. Ejecutar validación manual de ocupación regularmente (mínimo mensual).
2. Mantener log documentado de todas las correcciones aplicadas.
3. Monitorizar sensores que requieren recalibración frecuente.
4. Planificar reemplazo de sensores según vida útil estimada.
5. Coordinar mantenimiento con horarios de bajo uso de aparcamiento.
