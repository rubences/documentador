# 4.8 Actuación 8: Sistema de Sensorización Ambiental - Administración

## 4.8.1 Objeto del manual de administración

La presente sección describe las tareas de administración técnica del sistema de sensorización
ambiental, incluyendo gestión de estaciones, configuración de sensores y gestión de alertas
de calidad ambiental.

## 4.8.2 Perfiles de administración

| Perfil | Responsabilidades |
| --- | --- |
| Administrador de estaciones | Configuración de equipamiento y nodos de medición |
| Responsable de parámetros ambientales | Definición de umbrales y políticas de alertado |
| Técnico de mantenimiento | Control preventivo y reparación de sensores |

## 4.8.3 Acceso al módulo administrativo ambiental

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la consola de administración del sistema ambiental. |
| 2 | Iniciar sesión con credenciales de administrador. |
| 3 | Verificar acceso a configuración de estaciones y parámetros. |
| 4 | Confirmar disponibilidad de herramientas de análisis de tendencias. |

## 4.8.4 Gestión de estaciones de medición

| Acción | Instrucción |
| --- | --- |
| Registrar nueva estación | Documentar ubicación, altitud, exposición y equipamiento. |
| Instalar sensores | Configurar cada sensor con sus parámetros técnicos. |
| Definir rango de operación | Establecer limites físicos de medición para cada parámetro. |
| Validar calibración | Verificar que lecturas son consistentes con valores de referencia. |

## 4.8.5 Configuración de parámetros ambientales

| Acción | Instrucción |
| --- | --- |
| Definir parámetros a medir | Seleccionar qué variables ambientales monitorear (aire, ruido, etc.). |
| Establecer frecuencia de medición | Definir intervalo de lectura de sensores. |
| Configurar unidades de medida | Establecer sistema de unidades coherente (métrico, internacional). |
| Validar fórmulas de conversión | Verificar que cálculos derivados son correctos. |

## 4.8.6 Definición de alertas ambientales

| Acción | Instrucción |
| --- | --- |
| Crear umbral de alerta | Definir valor máximo o mínimo que dispara notificación. |
| Asignar severidad | Clasif icar alerta por importancia (bajo, medio, alto). |
| Configurar canales de escalado | Establecer protocolo de notificación según severidad. |
| Definir acciones automáticas | Configurar si alerta dispara cambios operacionales. |

## 4.8.7 Mantenimiento preventivo de equipamiento

| Acción | Instrucción |
| --- | --- |
| Programar calibración periódica | Agendar recalibración de sensores según especificación. |
| Revisar estado de batería | Monitorizar nivel energético en equipos inalámbricos. |
| Planificar reemplazo | Agendar cambio de sensores al final de vida útil. |
| Documentar intervenciones | Registrar mantenimiento realizado y resultados. |

## 4.8.8 Análisis de datos ambientales

| Acción | Instrucción |
| --- | --- |
| Generar informe de calidad | Crear resumen de parámetros ambientales por período. |
| Analizar tendencias | Identificar evolución temporal de indicadores clave. |
| Comparar con normativa | Validar cumplimiento de límites legales establecidos. |
| Exportar datos para reportes públicos | Preparar información para difusión municipal. |

## 4.8.9 Validación y calidad de datos

| Acción | Instrucción |
| --- | --- |
| Revisar lecturas anómalas | Identificar valores fuera de rango esperado. |
| Ejecutar validación cruzada | Comparar lecturas de sensores próximos para coherencia. |
| Aplicar correcciones | Ajustar valores identificados como erróneos tras análisis. |
| Documentar anomalías | Registrar problemas encontrados y acciones tomadas. |

## 4.8.10 Gestión de incidencias administrativas

| Incidencia | Actuación recomendada |
| --- | --- |
| Sensor no reporta datos | Revisar conectividad, batería y calibración del dispositivo. |
| Valor fuera de rango esperado | Validar en campo y comprobar si es evento real o mal funcionamiento. |
| Alerta no se genera | Verificar umbral configurado y estado del sistema de alertado. |
| Deterioro progresivo de lecturas | Planificar recalibración o reemplazo del sensor. |

## 4.8.11 Buenas prácticas administrativas

1. Ejecutar auditoría trimestral de precisión de mediciones.
2. Mantener log de calibraciones y mantenimiento preventivo realizado.
3. Revisar regularmente coherencia entre estaciones cercanas.
4. Documentar cambios en umbrales o parámetros monitoreados.
5. Realizar validaciones de campo mensualmente en al menos una estación.
