# 3.11 Actuación 11: Tarjeta Turística de San Lorenzo de El Escorial

## 3.11.1 Objeto del manual de usuario

La presente sección describe el uso funcional de la Tarjeta Turística por parte de personal
municipal y agentes autorizados para alta, validación, consulta de uso y gestión de incidencias.

## 3.11.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Gestor municipal | Administración de campañas y condiciones de uso |
| Operador de atención | Alta de usuarios y soporte funcional básico |
| Punto colaborador | Validación de beneficios y registro de uso |

## 3.11.3 Acceso al entorno de gestión

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al módulo de gestión de Tarjeta Turística. |
| 2 | Iniciar sesión con perfil autorizado. |
| 3 | Verificar disponibilidad de catálogos de beneficios activos. |
| 4 | Confirmar conectividad con sistemas de validación habilitados. |

## 3.11.4 Operaciones de usuario

| Acción | Instrucción |
| --- | --- |
| Dar de alta tarjeta | Registrar datos requeridos y activar la tarjeta. |
| Consultar estado | Verificar vigencia, condiciones y uso acumulado. |
| Aplicar beneficio | Validar derecho del usuario en punto colaborador. |
| Gestionar renovación | Actualizar periodo de validez según política definida. |
| Desactivar tarjeta | Bloquear uso por fin de vigencia o incidencia. |

## 3.11.5 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| Tarjeta no válida en un punto | Revisar vigencia, estado y sincronización de beneficios. |
| Beneficio no aplicado correctamente | Verificar reglas de campaña y condiciones de elegibilidad. |
| Error en datos del titular | Corregir información según permisos y registrar cambio. |
| Duplicidad de registro | Consolidar expediente y dejar trazabilidad de la actuación. |

## 3.11.6 Buenas prácticas

1. Mantener actualizado el catálogo de beneficios con fechas de vigencia.
2. Revisar periódicamente integridad de datos de usuarios activos.
3. Registrar incidencias recurrentes para mejora del proceso de validación.
4. Coordinar cambios de campaña con áreas de turismo y comercio.
