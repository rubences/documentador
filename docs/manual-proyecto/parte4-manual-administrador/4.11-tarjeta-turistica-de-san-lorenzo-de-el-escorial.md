# 4.11 Actuación 11: Tarjeta Turística de San Lorenzo de El Escorial - Administración

## 4.11.1 Objeto del manual de administración

La presente sección describe las tareas de administración técnica de la Tarjeta Turística,
incluyendo gestión de beneficiarios, configuración de beneficios y auditoría de transacciones.

## 4.11.2 Perfiles de administración

| Perfil | Responsabilidades |
| --- | --- |
| Administrador de tarjeta | Configuración de sistema y parámetros de beneficio |
| Gestor de campañas | Definición de promociones y condicionales de elegibilidad |
| Auditor de transacciones | Revisión de uso y validación de consistencia |

## 4.11.3 Acceso al módulo administrativo de tarjeta

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la consola de administración de Tarjeta Turística. |
| 2 | Iniciar sesión con credenciales de administrador. |
| 3 | Verificar acceso a gestión de beneficios y campañas. |
| 4 | Confirmar disponibilidad de herramientas de auditoría. |

## 4.11.4 Gestión de beneficiarios

| Acción | Instrucción |
| --- | --- |
| Crear beneficiario | Registrar datos de usuario en sistema. |
| Asignar tarjeta física | Vincular ID de tarjeta al beneficiario en base de datos. |
| Gestionar datos | Actualizar o corregir información personal según solicitud. |
| Desactivar beneficiario | Bloquear acceso por fin de vigencia o solicitud de usuario. |

## 4.11.5 Configuración de beneficios y promociones

| Acción | Instrucción |
| --- | --- |
| Crear nuevo beneficio | Definir descuento, cantidad de usos y ubicaciones participantes. |
| Establecer condiciones | Configurar elegibilidad (edad, residencia, otra). |
| Asignar vigencia | Establecer fecha de inicio y fin de validez del beneficio. |
| Validar integridad | Verificar que configuración no crea conflictos lógicos. |

## 4.11.6 Gestión de puntos colaboradores

| Acción | Instrucción |
| --- | --- |
| Registrar punto colaborador | Documentar comercio o servicio participante. |
| Asignar beneficios | Vincular promociones que ofrece el establecimiento. |
| Configurar acceso | Definir credenciales de validación para punto colaborador. |
| Auditar participación | Revisar número de transacciones por colaborador. |

## 4.11.7 Gestión de transacciones y uso

| Acción | Instrucción |
| --- | --- |
| Consultar transacciones | Revisar log de uso de tarjeta por beneficiario. |
| Validar consistencia | Verificar que transacciones corresponden a beneficios vigentes. |
| Corregir errores | Ajustar transacciones duplicadas o registradas incorrectamente. |
| Generar informe de uso | Crear resumen de promociones más utilizadas. |

## 4.11.8 Auditoría y conformidad

| Acción | Instrucción |
| --- | --- |
| Revisar acceso a puntos | Auditar quién accedió a cada punto colaborador y cuándo. |
| Validar descuentos aplicados | Verificar que montos otorgados corresponden a beneficios activos. |
| Revisar cambios de configuración | Acceder a log de modificaciones de beneficios. |
| Generar informe de compliance | Documentar conformidad con regulaciones de descuentos. |

## 4.11.9 Reportes y análisis

| Acción | Instrucción |
| --- | --- |
| Generar informe de adopción | Calcular número de beneficiarios activos e inactivos. |
| Analizar impacto | Medir efectividad de tarjeta en dinamización comercial. |
| Comparar temporadas | Evaluar diferencia en uso entre periodos turísticos altos y bajos. |
| Exportar datos para análisis | Proporcionar información a áreas de turismo para evaluación. |

## 4.11.10 Gestión de incidencias administrativas

| Incidencia | Actuación recomendada |
| --- | --- |
| Beneficiario sin acceso | Verificar estado de cuenta y activación de tarjeta. |
| Descuento no se aplica | Comprobar vigencia de beneficio y compatibilidad con punto. |
| Transacción duplicada | Auditar e invertir transacción duplicada si procede. |
| Punto colaborador reporta error | Revisar configuración de acceso y validar consistencia de datos. |

## 4.11.11 Buenas prácticas administrativas

1. Revisar mensualmente actividad de beneficiarios inactivos.
2. Documentar todos los cambios en configuración de beneficios.
3. Realizar auditoría trimestral de transacciones y consistencia.
4. Mantener comunicación con puntos colaboradores sobre actualizaciones.
5. Generar reportes de impacto regularmente para evaluación de ROI.
