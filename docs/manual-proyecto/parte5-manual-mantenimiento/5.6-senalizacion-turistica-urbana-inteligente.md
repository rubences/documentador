# 5.6 Actuación 6: Señalización Turística Urbana Inteligente - Mantenimiento

## 5.6.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo del
sistema de señalización turística inteligente, incluyendo reemplazo de baterías, validación
de conectividad y mantenimiento físico de dispositivos.

## 5.6.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Beacons BLE | Reemplazo de batería, validación de transmisión | Semestral |
| Códigos QR impresos | Inspección visual, limpieza | Trimestral |
| Lectores NFC | Validación de funcionamiento | Mensual |
| Soportes físicos | Inspección estructural, limpieza | Trimestral |
| Sistemas de energía | Validación de carga y conectores | Mensual |

## 5.6.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación de transmisión BLE | Verificar alcance de beacon en campo | Mensual | Técnico en campo |
| Inspección de baterías | Revisar nivel de energía por consola | Mensual | Equipo técnico |
| Limpieza de dispositivos | Remover suciedad y humedad | Trimestral | Equipo mantenimiento |
| Test de códigos QR | Escanear códigos para validar funcionamiento | Mensual | QA |
| Auditoría de dispositivos | Inventario completo de beacons activos | Semestral | Administrador |

## 5.6.4 Procedimiento de reemplazo de batería en beacon

| Paso | Instrucción |
| --- | --- |
| 1 | Identificar beacon con batería baja desde consola. |
| 2 | Registrar ubicación física y referencia en sistema. |
| 3 | Acceder al beacon (puede requerir escalera o herramientas). |
| 4 | Abrir carcasa utilizando procedimiento de seguridad. |
| 5 | Extraer batería agotada. |
| 6 | Instalar batería nueva del tipo correcto. |
| 7 | Cerrar carcasa y validar funcionamiento. |
| 8 | Registrar reemplazo de batería en sistema. |
| 9 | Documenta fecha, hora y técnico responsable. |

## 5.6.5 Validación de cobertura y alcance

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Test en campo | Validar alcance real del beacon en ubicación | Mensual |
| Prueba de distancia | Medir alcance máximo de transmisión | Trimestral |
| Test de obstáculos | Validar si edificios/vegetación afecta alcance | Trimestral |
| Documentación de resultados | Registrar mediciones en mapa de cobertura | Continua |

## 5.6.6 Mantenimiento de códigos QR

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Inspección visual | Verificar código no está dañado o decolorado | Mensual |
| Limpieza de superficies | Remover suciedad que afecte lectura | Trimestral |
| Validación de escaneo | Probar código con múltiples dispositivos | Mensual |
| Replacement de código dañado | Sustituir código si lectura no es posible | As-needed |

## 5.6.7 Procedimiento de reparación de fallos

| Tipo de falla | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Beacon no transmite | Revisar batería y conectar dispositivo | Reemplazar batería o beacon |
| QR no se escanea | Inspeccionar código e intentar limpiar | Reemplazar si está deteriorado |
| NFC no responde | Validar conectividad del chip | Reparar o reemplazar dispositivo |
| Soporte físico inestable | Inspeccionar anclaje | Reforzar o reemplazar soporte |

## 5.6.10 Buenas prácticas de mantenimiento

1. Ejecutar reemplazo de baterías de forma sistemática según vida útil.
2. Mantener stock de baterías de reemplazo en la cantidad adecuada.
3. Documentar cada intervención técnica en sistema centralizado.
4. Realizar auditoría de cobertura regularmente para identificar puntos muertos.
5. Comunicar mantenimiento planificado a través de consola administrativa.
