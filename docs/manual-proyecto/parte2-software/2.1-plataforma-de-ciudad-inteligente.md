# 2.1 Actuación 1: Plataforma de Ciudad Inteligente (PCI)

## 2.1.1 Descripción general de la solución software

La Plataforma de Ciudad Inteligente constituye el núcleo lógico del sistema y la base software
sobre la que se integran, procesan, explotan y publican los datos procedentes del resto de
actuaciones del contrato.

Conforme al Pliego, la solución debe responder a una arquitectura multicapa alineada con la
UNE 178104, con componentes abiertos, escalables, mantenibles y orientados a la
interoperabilidad, el análisis avanzado, la visualización y la automatización operativa.
