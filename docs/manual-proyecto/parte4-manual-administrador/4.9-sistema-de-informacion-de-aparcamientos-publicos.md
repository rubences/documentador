# 4.9 Actuación 9: Sistema de Información de Aparcamientos Públicos - Administración

## 4.9.1 Objeto del manual de administración

La presente sección describe las tareas de administración técnica del sistema de información
de aparcamientos públicos, incluyendo configuración de sensores, parámetros de conteo y
control de acceso administrativo.

## 4.9.2 Perfiles de administración

| Perfil | Responsabilidades |
| --- | --- |
| Administrador del sistema de aparcamiento | Configuración de sensores y zonas de conteo |
| Responsable de integridad de datos | Validación y corrección de desajustes de ocupación |
| Técnico de operaciones | Mantenimiento de infraestructura sensorial |

## 4.9.3 Acceso al módulo administrativo de aparcamientos

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la consola de administración del sistema de aparcamientos. |
| 2 | Iniciar sesión con credenciales de administrador. |
| 3 | Verificar acceso a configuración de aparcamientos y módulo de corrección. |
| 4 | Confirmar disponibilidad de herramientas de auditoría y reportes. |

## 4.9.4 Configuración de aparcamientos

| Acción | Instrucción |
| --- | --- |
| Registrar nuevo aparcamiento | Documentar nombre, ubicación, tipo (superficie/soterrado) y capacidad total. |
| Configurar zonas de conteo | Definir áreas internas para seguimiento de ocupación. |
| Instalar sensores | Ubicar detectores de presencia/magnetismo en entradas y salidas. |
| Validar comunicación | Verificar que todos los sensores envían datos correctamente. |

## 4.9.5 Configuración de parámetros de conteo

| Acción | Instrucción |
| --- | --- |
| Establecer capacidad total | Definir número máximo de plazas disponibles. |
| Configurar distancia de sensor | Ajustar sensibilidad según distancia entre líneas de detección. |
| Definir tiempo de debounce | Establecer retraso para evitar conteos duplicados. |
| Validar lógica de conteo | Verificar que entradas + salidas son coherentes. |

## 4.9.6 Publicación de disponibilidad

| Acción | Instrucción |
| --- | --- |
| Configurar actualización de paneles | Establecer sincronización con cartelería y web municipal. |
| Definir actualización a aplicación | Validar que app muestra información en tiempo real. |
| Configurar umbrales de difusión | Establecer si se publica disponibilidad total, por zona o solo ocupado. |
| Auditar propagación | Verificar que cambios de ocupación se reflejan en todos los canales. |

## 4.9.7 Gestión de correcciones administrativas

| Acción | Instrucción |
| --- | --- |
| Acceder a módulo de corrección | Autenticarse con perfil autorizado y mecanismo de firma. |
| Registrar desajuste observado | Documentar discrepancia encontrada en validación in-situ. |
| Aplicar corrección manual | Ajustar ocupación al valor real observado. |
| Guardar cambio con auditoría | Registrar quién, cuándo y por qué se realizó corrección. |

## 4.9.8 Análisis de datos de aparcamiento

| Acción | Instrucción |
| --- | --- |
| Generar informe de ocupación | Crear resumen de tasas de ocupación por período. |
| Analizar patrones de uso | Identificar horas punta y periodos de baja ocupación. |
| Comparar aparcamientos | Evaluar rendimiento relativo entre diferentes ubicaciones. |
| Exportar datos para planificación | Proporcionar información a áreas de movilidad municipal. |

## 4.9.9 Mantenimiento de infraestructura

| Acción | Instrucción |
| --- | --- |
| Revisar estado de sensores | Verificar funcionamiento correcto en pruebas periódicas. |
| Registrar fallos recurrentes | Identificar sensores con alta tasa de error. |
| Planificar mantenimiento | Agendar reparación o reemplazo de equipamiento defectuoso. |
| Documentar intervenciones | Registrar mantenimiento realizado y validación post-reparación. |

## 4.9.10 Gestión de incidencias administrativas

| Incidencia | Actuación recomendada |
| --- | --- |
| Ocupación no coincide con realidad | Realizar validación in-situ y aplicar corrección si es necesario. |
| Sensor da lecturas erráticas | Revisar calibración y posible desajuste físico del dispositivo. |
| No se actualiza disponibilidad en canal | Verificar conectividad y sincronización con sistema central. |
| Múltiples correcciones en aparcamiento | Analizar patrón e investigar causa raíz de desajustes. |

## 4.9.11 Buenas prácticas administrativas

1. Realizar validación de ocupación semanal en al menos un aparcamiento.
2. Mantener log de todas las correcciones manuales realizadas.
3. Revisar mensualmente aparcamientos con mayor número de desajustes.
4. Documentar cambios en configuración de sensores o umbrales.
5. Ejecutar auditoría trimestral de coherencia entre conteos e integridad. |
