# 2.9 Actuación 9: Sistema de Información de Aparcamientos Públicos

## 2.9.1 Descripción general de la solución software

La actuación contempla un sistema software para el control de ocupación de aparcamientos
públicos, cálculo de plazas libres, publicación de información en tiempo real y corrección
manual segura de errores de conteo.

Conforme al Pliego, la solución debe operar como webservice basado en software libre,
instalarse en un servidor municipal, exponer datos por HTTP y conectarse con el Gestor de
Contenidos para difusión multicanal.
