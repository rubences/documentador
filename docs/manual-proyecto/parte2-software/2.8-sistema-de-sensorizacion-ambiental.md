# 2.8 Actuación 8: Sistema de Sensorización Ambiental

## 2.8.1 Descripción general de la solución software

La actuación contempla un sistema software de gestión y monitorización de sensores
ambientales, integrado nativamente con la Plataforma de Ciudad Inteligente y con capacidad
para procesar, visualizar, alertar y explotar geográficamente los datos captados por las
estaciones instaladas.

Conforme al Pliego, la solución debe incluir programación y procesamiento en borde,
monitorización centralizada, paneles de indicadores y operación autónoma ante pérdida de
conectividad con la Plataforma.
