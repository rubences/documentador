# 2.3 Actuación 3: Renovación de la Web de Turismo

## 2.3.1 Descripción general de la solución software

La actuación contempla el diseño, desarrollo y puesta en producción de un nuevo portal web
turístico del municipio, alineado con la identidad municipal, accesible, multiidioma,
orientado al posicionamiento orgánico y conectado con el gestor de contenidos y la Plataforma
de Ciudad Inteligente.

Conforme al Pliego, el portal debe ser una solución llave en mano, plenamente funcional,
responsive, segura y preparada para interacción continua con turistas y residentes antes,
durante y después de la visita.
