# 5.5 Actuación 5: Cartelería Digital Centralizada - Mantenimiento

## 5.5.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo del
sistema de cartelería digital centralizada, incluyendo mantenimiento de paneles, actualización
de firmware y gestión de fallas de equipamiento.

## 5.5.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Pantallas digitales | Limpieza, validación visual | Mensual |
| Controladores de panel | Firmware, monitorización de fallos | Trimestral |
| Conexiones de red | Validación de enlaces, diagnostico | Semanal |
| Fuentes de alimentación | Inspección visual, test de carga | Trimestral |
| Infraestructura de distribución | Monitorización de ancho de banda | Diaria |

## 5.5.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación visual de paneles | Revisar pantalla funciona correctamente | Mensual | Técnico en campo |
| Limpieza de pantallas | Remover polvo y suciedad acumulada | Trimestral | Equipo mantenimiento |
| Validación de brillo | Comprobar que nivel de brillo es adecuado | Mensual | Monitorización |
| Test de conectividad | Verificar comunicación desde consola central | Semanal | Equipo técnico |
| Auditoría de firmware | Verificar que versión es la más reciente | Trimestral | Administrador |

## 5.5.4 Procedimiento de limpieza de pantallas

| Paso | Instrucción |
| --- | --- |
| 1 | Programar limpieza en horario de baja demanda. |
| 2 | Pausar contenido en panel a limpiar. |
| 3 | Utilizar materiales apropiados (alcohol isopropílico, paño sin pelusas). |
| 4 | Limpiar pantalla cuidadosamente sin aplicar presión excesiva. |
| 5 | Inspeccionar visualmente por defectos. |
| 6 | Reanudar contenido en panel. |
| 7 | Documentar fecha y resultado de limpieza. |

## 5.5.5 Mantenimiento de controladores y firmware

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Monitorización de logs | Revisar eventos de error en controlador | Semanal |
| Validación de temperatura | Comprobar que temperatura de operación es normal | Mensual |
| Actualización de firmware | Instalar versión más reciente de software | Trimestral |
| Test post-actualización | Validar que panel responde correctamente | Post-actualización |
| Backup de configuración | Guardar parámetros antes de cambios | Pre-actualización |

## 5.5.6 Procedimiento de actualización de firmware

| Paso | Instrucción |
| --- | --- |
| 1 | Descargar nueva versión de firmware. |
| 2 | Realizar backup de configuración actual. |
| 3 | Desactivar transmisión de contenido a panel. |
| 4 | Ejecutar actualización remota desde consola central. |
| 5 | Validar que actualización completó correctamente. |
| 6 | Verificar que panel está operacional. |
| 7 | Reactivar transmisión de contenido. |
| 8 | Documentar versión actualizada y fecha. |

## 5.5.7 Gestión de fallas de equipamiento

| Tipo de falla | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Pantalla no enciende | Revisar alimentación eléctrica y conectores | Reparar conexión o sustituir fuente |
| Contenido no se actualiza | Test de conectividad de red | Revisar cable de red o reiniciar panel |
| Píxeles muertos | Inspección visual del panel | Sustituir pantalla si falla es crítica |
| Controlador no responde | Verificar conectividad del equipo | Reiniciar controlador o sustituir |
| Problemas de visualización | Ajustar brillo, contraste y resolución | Recalibrar o reemplazar pantalla |

## 5.5.8 Plan de sustitución de componentes

| Componente | Vida útil esperada | Política de sustitución |
| --- | --- | --- |
| Pantalla | 5-7 años | Reemplazar al fin de vida o por avería |
| Controlador | 7-10 años | Reemplazar si firmware no es compatible |
| Fuente de alimentación | 3-5 años | Reemplazar preventivamente al final de vida |
| Cables y conectores | 5-7 años | Reemplazar si corrosión o daño visible |

## 5.5.9 Auditoría de estado del sistema

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Inventario de paneles | Verificar número y estado de equipamiento | Semestral |
| Análisis de fallos | Revisar paneles con mayor número de incidencias | Trimestral |
| Validación de cobertura | Confirmar que todos los puntos están operativos | Mensual |
| Generación de reportes | Crear resumen de estado de cartelería | Mensual |

## 5.5.10 Buenas prácticas de mantenimiento

1. Ejecutar limpieza de pantallas de forma periódica y coordinada.
2. Mantener stock de componentes de repuesto críticos.
3. Documentar todas las intervenciones en cada panel.
4. Realizar planificación de renovación basada en vida útil de equipamiento.
5. Comunicar mantenimiento planificado a través de consola de control.
