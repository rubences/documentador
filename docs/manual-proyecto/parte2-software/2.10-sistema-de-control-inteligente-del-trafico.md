# 2.10 Actuación 10: Sistema de Control Inteligente del Tráfico

## 2.10.1 Descripción general de la solución software

La actuación contempla una plataforma software para la gestión inteligente del tráfico urbano,
centrada en el reconocimiento automático de matrículas, el conteo de vehículos, la trazabilidad
de recorridos, la generación de estadísticas y la asistencia operativa a la Policía Local.

Conforme al Pliego, la solución debe operar sobre un servidor local, ser compatible con la
Plataforma de Ciudad Inteligente, procesar información en tiempo real y facilitar futuras
ampliaciones mediante API abierta incluida en el contrato.
