# 3.12 Actuación 12: Dinamización e Impulso Digital y Sostenible del Empresariado Turístico y Comercial

## 3.12.1 Objeto del manual de usuario

La presente sección describe el uso funcional del entorno de dinamización digital dirigido a
empresas y comercios locales, incluyendo gestión de contenidos, campañas y seguimiento de
participación en acciones de promoción.

## 3.12.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Administrador municipal | Coordinación de convocatorias y campañas |
| Técnico dinamizador | Acompañamiento y soporte a empresas adheridas |
| Empresa participante | Actualización de información y uso de recursos digitales |

## 3.12.3 Acceso al entorno de trabajo

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al portal o módulo de dinamización habilitado. |
| 2 | Iniciar sesión con el perfil correspondiente. |
| 3 | Revisar calendario de acciones y campañas vigentes. |
| 4 | Confirmar disponibilidad de recursos de difusión y seguimiento. |

## 3.12.4 Operación funcional

| Acción | Instrucción |
| --- | --- |
| Crear campaña de dinamización | Definir objetivo, alcance, fechas y colectivos destinatarios. |
| Publicar convocatoria | Difundir requisitos y plazos a empresas del municipio. |
| Registrar participantes | Validar alta y estado de adhesión de establecimientos. |
| Actualizar fichas de negocio | Mantener datos comerciales y de contacto vigentes. |
| Medir resultados | Consultar participación, alcance y métricas de actividad. |

## 3.12.5 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| Empresa no puede acceder | Verificar credenciales, estado de alta y permisos asignados. |
| Datos comerciales no se actualizan | Revisar validaciones del formulario y guardar cambios de nuevo. |
| Campaña no visible | Comprobar fechas de vigencia, estado de publicación y filtros. |
| Métricas incompletas | Revisar configuración de seguimiento y origen de datos. |

## 3.12.6 Buenas prácticas

1. Comunicar con antelación plazos y criterios de participación.
2. Mantener calidad y actualización de la información de establecimientos.
3. Homogeneizar indicadores para comparar resultados entre campañas.
4. Documentar lecciones aprendidas tras cada acción de dinamización.
