# 3.3 Actuación 3: Renovación de la Web de Turismo

## 3.3.1 Objeto del manual de usuario

La presente sección describe el uso básico del portal web turístico por parte de usuarios
finales, visitantes y personal municipal con acceso funcional a formularios y servicios de
consulta.

## 3.3.2 Acceso al portal

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al portal web turístico oficial del municipio. |
| 2 | Seleccionar idioma, en su caso, desde el selector habilitado en la cabecera o menú principal. |
| 3 | Navegar por secciones, buscador, mapas o formularios según la necesidad del usuario. |

## 3.3.3 Navegación básica del portal

| Acción | Descripción |
| --- | --- |
| Consultar noticias y eventos | Acceder a las secciones informativas para revisar contenidos actualizados. |
| Localizar recursos turísticos | Utilizar buscadores, mapas o menús de navegación jerárquicos. |
| Revisar rutas | Consultar rutas turísticas, rutas de compras u otras rutas disponibles. |
| Consultar agenda | Revisar actividades, eventos y programación del destino. |
| Usar formularios | Realizar solicitudes de información, reservas u otras gestiones habilitadas. |

## 3.3.4 Funcionalidades de usuario

| Funcionalidad | Instrucción de uso |
| --- | --- |
| Buscador general | Introducir términos de búsqueda para localizar contenidos y secciones. |
| Mapa del sitio | Consultar la estructura jerárquica completa del portal. |
| Geolocalización en mapas | Visualizar recursos y puntos de interés sobre mapa cuando esté disponible. |
| Compartir contenido | Utilizar la opción de compartir en redes sociales o canales compatibles. |
| Newsletter | Suscribirse mediante el formulario habilitado, si está disponible. |
| Chatbot | Realizar consultas rápidas si el módulo se encuentra implantado. |

## 3.3.5 Registro y cuenta de usuario

| Acción | Instrucción |
| --- | --- |
| Registrarse | Completar el procedimiento de alta si el usuario desea disponer de cuenta. |
| Iniciar sesión | Acceder con credenciales en los espacios que lo requieran. |
| Solicitar baja | Ejercer la eliminación de la cuenta y de los datos personales facilitados. |

## 3.3.6 Accesibilidad y uso recomendado

1. Utilizar el selector de idioma cuando el contenido esté disponible en varias lenguas.
2. Hacer uso de los menús simplificados, buscador y mapa del sitio para localizar recursos con mayor rapidez.
3. En caso de uso con tecnologías de apoyo, emplear los controles accesibles del portal y los textos alternativos disponibles.
4. Comprobar la información legal y condiciones del servicio antes de completar formularios o suscripciones.

## 3.3.7 Incidencias frecuentes de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| Una sección no carga contenido | Actualizar la página y revisar conectividad del navegador. |
| No se encuentra un recurso | Utilizar buscador general, filtros o mapa del sitio. |
| Un formulario no se envía | Revisar campos obligatorios y volver a intentar el envío. |
| El idioma no cambia en un contenido | Comprobar si el contenido está disponible en esa lengua. |
