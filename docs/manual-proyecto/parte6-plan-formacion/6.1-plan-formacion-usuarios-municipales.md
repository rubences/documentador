# 6.1 Plan de Formación para Usuarios Municipales (Editores de Contenidos)

## 6.1.1 Objeto del plan de formación

La presente sección describe el plan de formación para usuarios municipales con capacidad
de edición, quienes introducirán y manejarán contenidos en las distintas actuaciones del
proyecto. Este plan garantiza que los usuarios adquieran competencias autónomas en las 13
actuaciones.

## 6.1.2 Perfil de participantes

| Característica | Descripción |
| --- | --- |
| Perfil objetivo | Usuarios municipales responsables de introducir y gestionar contenidos |
| Nivel técnico inicial | Básico a intermedio |
| Requisitos previos | Conocimientos básicos de ofimática y navegación web |
| Número aproximado de participantes | 5–15 usuarios |
| Dedicación esperada | 2–3 horas por actuación + práctica posterior |

## 6.1.3 Objetivos de formación

1. Manejar de forma autónoma todas las herramientas de edición de contenidos.
2. Comprender los flujos de trabajo para cada actuación.
3. Aplicar buenas prácticas de seguridad y control de cambios.
4. Resolver problemas comunes sin necesidad de soporte técnico.
5. Contribuir a mantener la calidad y actualización de contenidos.

| 6.3 Web de Turismo | Gestión de páginas, banners, menús, integración con CMS | 2 h | Editar página existente |
| 6.4 Apps de Turismo | Gestión de contenidos en app, notificaciones push básicas | 2 h | Enviar notificación |
| 6.5 Cartelería Digital | Plantillas de cartelería, programación de contenidos, calendario | 1,5 h | Crear cartel temporal |
| 6.6 Señalización Turística | Gestión de beacons, contenidos de puntos de interés | 1,5 h | Registrar punto de interés |
| 6.7 Monitorización de Flujos | Consulta de datos de flujos, generación de reportes | 1,5 h | Crear reporte básico |
| 6.8 Sensorización Ambiental | Consulta de datos ambientales, alertas, histórico | 1 h | Revisar datos históricos |
| 6.9 Aparcamientos | Consulta de disponibilidad, tarifas (si aplica), reportes | 1 h | Consultar ocupación |

## 6.1.5 Metodología de formación

| Fase | Descripción | Duración |
| --- | --- | --- |
| Presentación inicial | Contexto del proyecto, objetivos generales, estructura de actuaciones | 2 h |
| Formación específica por actuación | Sesiones de 1–2,5 h con demostración y práctica guiada | 20 h |
| Práctica autónoma supervisada | Cada usuario realiza tareas reales bajo supervisión | 4–8 h |
| Evaluación final | Completar tareas de complejidad media de forma autónoma | 2 h |
| **Total estimado** | **Formación presencial + práctica** | **~27 horas** |

| --- | --- | --- |

## 6.1.8 Criterios de evaluación

| Criterio | Descripción | Evaluación |
| --- | --- | --- |
| Comprensión funcional | Usuario entiende flujos principales de cada actuación | Observación + Cuestionario |
| Autonomía operacional | Usuario puede realizar tareas sin asistencia | Prueba práctica observada |
| Manejo de errores | Usuario sabe identificar y reportar problemas | Simulación de problema |
| Seguridad | Usuario respeta permisos y no accede datos restringidos | Prueba de seguridad |
| Documentación | Usuario sabe dónde buscar ayuda y documentación | Conversación |

## 6.1.9 Evaluación de competencia final

Al completar la formación, cada usuario deberá ser capaz de:

- ✅ Crear, editar y publicar contenido en CMS y Web sin asistencia
- ✅ Programar contenido en cartelería digital para fecha/hora específica
- ✅ Registrar punto de interés con información turística completa
- ✅ Consultar datos de flujos, sensores, aparcamiento y tráfico
- ✅ Generar reportes básicos de actividad
- ✅ Crear beneficio/descuento en Tarjeta Turística
- ✅ Crear y lanzar campaña de dinamización digital
- ✅ Identificar y reportar problemas técnicos adecuadamente

**Se considerará completada cuando el usuario sea capaz de realizar estas tareas de forma autónoma y consistente.**

## 6.1.10 Plan de seguimiento post-formación

| Periodo | Actividad | Responsable |
| --- | --- | --- |
| Semana 1–4 | Soporte intensivo, respuesta a dudas en <4 horas | Adjudicatario |
| Mes 1–2 | Auditoría de contenidos creados, feedback constructivo | Ambos |
| Mes 3 | Sesión de soporte presencial (2 h) – dudas y mejoras | Adjudicatario |
| Mes 3–6 | Soporte estándar, respuesta a dudas en <1 día laboral | Adjudicatario |
| Mes 6 | Sesión de soporte presencial (2 h) – evaluación y temas avanzados | Adjudicatario |
| Mes 6+ | Soporte continuo según contrato de mantenimiento | Adjudicatario |
