# 2.7 Actuación 7: Monitorización de Flujos Turísticos

## 2.7.1 Descripción general de la solución software

La actuación contempla un sistema software para el control de aforo y movilidad peatonal y
ciclista, integrado con la Plataforma de Ciudad Inteligente y orientado a la obtención de datos
en tiempo real e históricos, cuadros de mando y alertas.

Conforme al Pliego, la solución debe procesar información procedente de cámaras y sensores,
generar estadísticas, soportar analítica histórica y ofrecer visualización y control mediante una
interfaz web o móvil.
