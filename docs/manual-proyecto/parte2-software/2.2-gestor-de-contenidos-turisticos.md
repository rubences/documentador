# 2.2 Actuación 2: Gestor de Contenidos Turísticos

## 2.2.1 Descripción general de la solución software

La actuación contempla la implantación de un gestor de contenidos turísticos para centralizar la
administración de contenidos municipales y su distribución hacia web, aplicación móvil,
señalización, beacons y cartelería digital.

Conforme al Pliego, el sistema debe estar basado en software libre, modular, escalable, alto
configurable y orientado a la gestión multicanal, multiidioma, georreferenciada y con
exposición mediante servicios web y API REST.
