# 5.8 Actuación 8: Sistema de Sensorización Ambiental - Mantenimiento

## 5.8.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo del
sistema de sensorización ambiental, incluyendo calibración periódica, validación de lecturas
y gestión de fallos de equipamiento.

## 5.8.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Sensores ambientales | Calibración, limpieza | Semestral |
| Estación de medición | Inspección estructural, limpieza | Trimestral |
| Conexión de energía | Validación de suministro | Mensual |
| Transmisión de datos | Validación de conectividad | Semanal |
| Base de datos de mediciones | Integridad, backup | Diaria |

## 5.8.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación de lecturas | Verificar que mediciones son dentro de rango esperado | Diaria | Equipo técnico |
| Inspección visual de estación | Revisar estado físico del equipamiento | Mensual | Técnico en campo |
| Limpieza de filtros | Remover polvo de filtros de aires si aplica | Trimestral | Equipo mantenimiento |
| Calibración de sensores | Ajustar sensores a valores de referencia | Semestral | Especialista |
| Auditoría de rendimiento | Revisar precisión global del sistema | Trimestral | Responsable ambiental |

## 5.8.4 Procedimiento de calibración de sensores

| Paso | Instrucción |
| --- | --- |
| 1 | Obtener solución o gas de calibración de referencia. |
| 2 | Documentar fecha, hora y responsable de calibración. |
| 3 | Acceder al sensor siguiendo procedimiento de seguridad. |
| 4 | Aplicar solución de calibración según especificación. |
| 5 | Registrar valor de lectura del sensor. |
| 6 | Comparar con valor esperado de referencia. |
| 7 | Ajustar sensor si diferencia > umbral de tolerancia. |
| 8 | Validar lectura post-ajuste. |
| 9 | Documentar calibración completada. |

## 5.8.5 Validación de lecturas de sensores

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Revisión de rangos | Verificar que lecturas están dentro de rango físico posible | Diaria |
| Análisis de outliers | Identificar mediciones anómalamente altas o bajas | Diaria |
| Comparación con estaciones cercanas | Validar si lecturas de estaciones proximas son coherentes | Semanal |
| Análisis de tendencias | Revisar si evolución temporal es realista | Semanal |

## 5.8.6 Mantenimiento de estaciones

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Limpieza de equipamiento | Remover polvo y contaminación | Trimestral |
| Inspección de soportes | Verificar que estructura es estable | Semestral |
| Validación de drenaje | Comprobar que agua no se acumula | Trimestral |
| Pintura y corrosión | Revisar si hay oxidación o daños | Semestral |

## 5.8.7 Procedimiento de reemplazo de sensores

| Paso | Instrucción |
| --- | --- |
| 1 | Identificar sensor a reemplazar según programación o fallo. |
| 2 | Realizar último backup de datos del sensor antiguo. |
| 3 | Desconectar sensor antiguo de forma segura. |
| 4 | Instalar nuevo sensor en posición original. |
| 5 | Realizar calibración inicial del nuevo sensor. |
| 6 | Validar que sensor reporta datos correctamente. |
| 7 | Documentar reemplazo de sensor. |

## 5.8.8 Gestión de fallos de equipamiento

| Tipo de falla | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Sensor no reporta datos | Revisar conectividad eléctrica y de comunicaciones | Reiniciar o reparar conexión |
| Lecturas fuera de rango | Recalibrar sensor o revisar si está dañado | Recalibrar o reemplazar |
| Lecturas erráticas | Validar fuentes de interferencia | Aislar interferencias o reparar sensor |
| Estación offline | Verificar energía y conectividad de red | Reiniciar equipo o reparar conexión |

## 5.8.9 Documentación de mantenimiento

| Documento | Contenido | Actualización |
| --- | --- | --- |
| Log de calibraciones | Fecha, sensor, valores, responsable | Cada calibración |
| Registro de reemplazos | Fecha, sensor antiguo/nuevo, razón | Cada reemplazo |
| Historial de fallos | Problemas reportados y resoluciones | Continua |
| Gráfico de precisión | Evolución de precisión de sensores en tiempo | Mensual |

## 5.8.10 Buenas prácticas de mantenimiento

1. Mantener calendario de calibración en base a vida útil del sensor.
2. Utilizar soluciones de calibración certificadas y documentadas.
3. Documentar todas las calibraciones y reemplazos realizados.
4. Ejecutar validación de precisión regularmente.
5. Planificar renovación de sensores según especificación de fabricante.
