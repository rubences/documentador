# 3.4 Actuación 4: Renovación y Realización de App's de Turismo

## 3.4.1 Objeto del manual de usuario

La presente sección describe el uso básico de la aplicación móvil turística por parte de turistas,
visitantes y usuarios registrados, tanto en consulta general como en funciones avanzadas de
rutas, favoritos y uso de balizas.

## 3.4.2 Instalación y acceso

| Paso | Instrucción |
| --- | --- |
| 1 | Descargar la aplicación desde la tienda oficial correspondiente al dispositivo. |
| 2 | Instalar la aplicación y aceptar únicamente los permisos necesarios para su funcionamiento. |
| 3 | Abrir la pantalla de bienvenida y seleccionar, si procede, idioma y preferencias básicas. |
| 4 | Acceder en modo consulta o iniciar registro/login si se desea disponer de servicios personalizados. |

## 3.4.3 Navegación principal

| Sección | Uso principal |
| --- | --- |
| Inicio | Acceso a noticias, rutas destacadas y rutas del usuario |
| Perfil | Gestión de datos básicos y preferencias |
| Mapa georreferenciado | Localización de hitos, balizas y navegación |
| Favoritos | Consulta y gestión de puntos guardados |
| Mis rutas | Consulta y gestión de rutas generadas o grabadas |
| Buscador | Localización de contenidos por criterios relevantes |

## 3.4.4 Uso de rutas turísticas

| Acción | Instrucción |
| --- | --- |
| Seleccionar ruta | Elegir entre rutas disponibles desde la sección correspondiente. |
| Iniciar navegación | Activar el guiado mediante los mapas nativos del dispositivo. |
| Consultar distancia y tiempo | Revisar los tiempos estimados al siguiente hito y a la finalización. |
| Interactuar con hitos | Al aproximarse a una baliza o tótem, consultar el contenido contextual asociado. |
| Guardar como favorita | Añadir la ruta o recurso a favoritos si la opción está disponible. |

## 3.4.5 Funcionalidades de usuario

| Funcionalidad | Instrucción de uso |
| --- | --- |
| Búsqueda de contenidos | Utilizar el buscador por categorías, recursos o términos libres. |
| Lectura QR/BIDI | Activar la función de lectura y escanear el código correspondiente. |
| Notificaciones PUSH | Gestionar consentimiento y recepción desde la configuración del dispositivo o de la app. |
| Compartir recurso | Utilizar la función nativa de compartición del sistema operativo. |
| Ayuda online | Consultar información funcional, legal y condiciones del servicio. |
| TTS / lectura accesible | Activar la funcionalidad de locución si está disponible en el idioma requerido. |

## 3.4.6 Registro y cuenta de usuario

| Acción | Instrucción |
| --- | --- |
| Registro | Completar alta con los datos solicitados por la aplicación. |
| Inicio de sesión | Acceder con credenciales en caso de usuario registrado. |
| Recuperación de contraseña | Utilizar la opción específica desde la pantalla de acceso. |
| Baja de usuario | Solicitar la eliminación del registro desde las opciones disponibles. |

## 3.4.7 Configuración personal

| Opción | Uso |
| --- | --- |
| Mapa por defecto | Selección del mapa de preferencia cuando la solución lo permita |
| Tamaño de letra | Ajuste de legibilidad |
| Idioma | Selección de idioma disponible |
| Preferencias de usuario | Ajustes funcionales definidos en la implantación |

## 3.4.8 Buenas prácticas de uso

1. Activar geolocalización solo cuando se desee usar rutas y contenidos contextuales.
2. Revisar notificaciones y preferencias de idioma al primer uso.
3. Utilizar el modo registro únicamente si se quieren conservar rutas, historial o datos personalizados.
4. Comprobar que el dispositivo dispone de conexión y permisos adecuados antes de iniciar una ruta.

## 3.4.9 Incidencias frecuentes de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| La ruta no se muestra en el mapa | Verificar geolocalización y conectividad del dispositivo. |
| No aparece contenido al pasar por una baliza | Comprobar Bluetooth, permisos y proximidad al hito. |
| No se reciben notificaciones | Revisar permisos PUSH en la aplicación y en el sistema operativo. |
| No funciona el acceso con usuario | Revisar credenciales o utilizar recuperación de contraseña. |
