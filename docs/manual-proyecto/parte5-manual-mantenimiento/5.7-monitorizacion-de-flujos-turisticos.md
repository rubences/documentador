# 5.7 Actuación 7: Monitorización de Flujos Turísticos - Mantenimiento

## 5.7.1 Objeto del manual de mantenimiento

La presente sección describe los procedimientos de mantenimiento preventivo y correctivo del
sistema de monitorización de flujos, incluyendo calibración de sensores, validación de datos
y gestión de fallos de equipamiento.

## 5.7.2 Equipamiento y componentes sujetos a mantenimiento

| Componente | Tipo de mantenimiento | Frecuencia |
| --- | --- | --- |
| Sensores de movimiento | Calibración, limpieza de óptica | Trimestral |
| Conectividad de nodos | Validación de enlaces de comunicación | Semanal |
| Base de datos de conteos | Integridad, optimización | Mensual |
| Servidores de análisis | Monitorización de rendimiento | Diaria |
| Fuentes de energía de nodos | Revisión de conectores y batería | Mensual |

## 5.7.3 Plan de mantenimiento preventivo

| Tarea | Descripción | Frecuencia | Responsable |
| --- | --- | --- | --- |
| Validación de nodos | Verificar que nodo reporta datos correctamente | Semanal | Equipo técnico |
| Calibración de sensores | Revisar que mediciones son precisas | Trimestral | Especialista |
| Limpiar óptica de sensor | Remover polvo de lentes de sensor | Trimestral | Técnico en campo |
| Test de comunicación | Validar conectividad del nodo | Semanal | Equipo técnico |
| Auditoría de datos | Verificar integridad de conteos | Mensual | Analista |

## 5.7.4 Procedimiento de calibración de sensores

| Paso | Instrucción |
| --- | --- |
| 1 | Realizar conteo manual de personas en zona de sensor. |
| 2 | Comparar conteo manual con dato del sensor. |
| 3 | Calcular diferencia porcentual (tolerancia: +/-5%). |
| 4 | Si diferencia > 5%, ajustar parámetros de sensor. |
| 5 | Limpiar óptica si lectura es anómalamente baja. |
| 6 | Repetir validación hasta que precisión sea aceptable. |
| 7 | Documentar calibración realizada. |

## 5.7.5 Mantenimiento de conectividad de nodos

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Validación de conexión de red | Verificar que nodo está online | Diaria (automática) |
| Test de latencia | Revisar tiempo de respuesta de nodo | Semanal |
| Inspección física de cables | Revisar conectores y cable no dañados | Trimestral |
| Validación de alimentación | Comprobar que nodo tiene energía | Diaria (automática) |
| Reinicio remoto si es necesario | Ejecutar reboot de nodo para recuperación | As-needed |

## 5.7.6 Gestión de fallos de sensores

| Tipo de falla | Diagnóstico | Acción correctiva |
| --- | --- | --- |
| Nodo offline | Verificar conectividad red y energía | Reiniciar nodo o reparar conexión |
| Conteos anómalamente altos | Revisar si sensor está saturado | Limpiar o reemplazar sensor |
| Conteos cero en horario activo | Validar si zona tiene tráfico real | Recalibrar o reparar sensor |
| Pérdida de comunicación intermitente | Test de conectividad de red | Revisar cables y interfaces |

## 5.7.7 Auditoría de calidad de datos

| Tarea | Descripción | Frecuencia |
| --- | --- | --- |
| Análisis de anomalías | Identificar conteos fuera de rango normal | Semanal |
| Validación de consistencia | Verificar que entradas ≈ salidas | Semanal |
| Comparación entre nodos | Evaluar si datos de nodos proximales son coherentes | Mensual |
| Detección de patrones | Identificar si datos muestran patrones sospechosos | Trimestral |

## 5.7.8 Procedimiento de validación en campo

| Paso | Instrucción |
| --- | --- |
| 1 | Seleccionar nodo para validación. |
| 2 | Establecer período de validación (p.ej. 1 hora). |
| 3 | Realizar conteo manual de personas en zona. |
| 4 | Registrar dato del sensor para mismo período. |
| 5 | Calcular precisión (contador/manual). |
| 6 | Documentar resultado. |
| 7 | Tomar acción correctiva si precisión < 95%. |

## 5.7.9 Mantenimiento de infraestructura

| Acción | Descripción | Frecuencia |
| --- | --- | --- |
| Inspección de cableado | Revisar estado de cables y conectores | Trimestral |
| Limpieza de zonas de sensor | Remover obstáculos que afecten detección | Mensual |
| Test de redundancia | Validar failover si nodo principal falla | Trimestral |
| Validación de sincronización horaria | Verificar que todos los nodos tienen hora consistente | Mensual |

## 5.7.10 Buenas prácticas de mantenimiento

1. Ejecutar calibración periódica de sensores según procedimiento estándar.
2. Documentar todas las calibraciones realizadas para auditoría.
3. Mantener log de fallos y tiempos de resolución.
4. Realizar validaciones en campo regularmente (mínimo trimestral).
5. Planificar renovación de sensores según vida útil estimada.
