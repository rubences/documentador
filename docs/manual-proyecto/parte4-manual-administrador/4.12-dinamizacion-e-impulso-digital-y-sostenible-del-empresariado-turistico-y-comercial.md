# 4.12 Actuación 12: Dinamización e Impulso Digital y Sostenible del Empresariado Turístico y Comercial - Administración

## 4.12.1 Objeto del manual de administración

La presente sección describe las tareas de administración técnica de la plataforma de
dinamización digital, incluyendo gestión de usuarios empresariales, campañas y seguimiento
de participación.

## 4.12.2 Perfiles de administración

| Perfil | Responsabilidades |
| --- | --- |
| Administrador de plataforma | Configuración técnica y gestión de acceso |
| Responsable de campañas | Diseño de acciones y seguimiento de adhesión |
| Analista de participación | Medición de resultados y elaboración de reportes |

## 4.12.3 Acceso al módulo administrativo de dinamización

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la consola de administración de dinamización digital. |
| 2 | Iniciar sesión con credenciales de administrador. |
| 3 | Verificar acceso a gestión de empresas y campañas. |
| 4 | Confirmar disponibilidad de herramientas de análisis. |

## 4.12.4 Gestión de usuarios empresariales

| Acción | Instrucción |
| --- | --- |
| Registrar empresa | Documentar datos de negocio y contacto. |
| Crear usuario de acceso | Asignar credenciales a responsable de empresa. |
| Asignar permisos | Definir qué áreas de contenido puede administrar la empresa. |
| Desactivar acceso | Bloquear empresa por fin de participación o solicitud. |

## 4.12.5 Configuración de campañas

| Acción | Instrucción |
| --- | --- |
| Crear convocatoria | Definir objetivo, fechas y requisitos de participación. |
| Publicar convocatoria | Difundir a empresas objetivo y habilitar inscripción. |
| Gestionar adhesiones | Revisar solicitudes y aprobar/rechazar participantes. |
| Configurar roles de participante | Definir qué puede ver y editar cada empresa adherida. |

## 4.12.6 Gestión de contenidos empresariales

| Acción | Instrucción |
| --- | --- |
| Configurar campos de perfil | Definir qué información solicitar a empresas. |
| Validar fichas de negocio | Revisar datos completados por empresas antes de publicación. |
| Gestionar actualizaciones | Procesar cambios en información empresarial. |
| Publicar información | Hacer visible ficha de empresa en canales habilitados. |

## 4.12.7 Seguimiento de participación

| Acción | Instrucción |
| --- | --- |
| Revisar estado de participante | Verificar que empresa cumple requisitos de adhesión. |
| Monitorizar actividad | Seguimiento de actualizaciones y acciones realizadas. |
| Registrar incumplimientos | Documentar si empresa no mantiene información actualizada. |
| Auditar acceso | Revisar cuándo y qué modificó cada empresa. |

## 4.12.8 Análisis de resultados

| Acción | Instrucción |
| --- | --- |
| Contar participantes | Calcular número de empresas adheridas a cada campaña. |
| Medir participación | Analizar nivel de actividad y actualización de fichas. |
| Evaluar impacto | Medir visibilidad y beneficio percibido por participantes. |
| Generar informe | Crear resumen de resultados de campaña. |

## 4.12.9 Auditoría y validación

| Acción | Instrucción |
| --- | --- |
| Revisar log de cambios | Acceder a historial de modificaciones por empresa. |
| Validar datos | Auditar integridad de información comercial. |
| Revisar permisos | Verificar que empresas no acceden a datos de competidores. |
| Generar certificado de cumplimiento | Documentar que campaña cumplió requisitos. |

## 4.12.10 Gestión de incidencias administrativas

| Incidencia | Actuación recomendada |
| --- | --- |
| Empresa no puede acceder | Verificar estado de cuenta y permisos asignados. |
| Datos no se actualizan | Revisar permisos de edición y validación de formulario. |
| Baja participación | Contactar empresas para entender barreras de participación. |
| Información inconsistente | Auditar y solicitar corrección a empresa responsable. |

## 4.12.11 Buenas prácticas administrativas

1. Comunicar fechas y requisitos de campañas con antelación suficiente.
2. Revisar periódicamente actualización de fichas de empresas activas.
3. Documentar lecciones aprendidas tras cada campaña.
4. Mantener comunicación regular con empresas participantes.
5. Generar reportes trimestrales de impacto para evaluación de estrategia.
