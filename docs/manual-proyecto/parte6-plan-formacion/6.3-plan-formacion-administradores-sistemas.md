# 6.3 Plan de Formación para Administradores de Sistemas

## 6.3.1 Objeto del plan de formación

La presente sección describe el plan de formación para administradores de sistemas, quienes
serán responsables de la administración técnica, seguridad, copias de seguridad y gestión
de incidencias críticas en una o más actuaciones del proyecto. Este plan garantiza
competencias avanzadas en operación y administración.

## 6.3.2 Perfil de participantes

| Característica | Descripción |
| --- | --- |
| Perfil objetivo | Administradores de sistemas municipales con experiencia previa |
| Nivel técnico inicial | Avanzado (Linux, Windows, redes, bases de datos) |
| Requisitos previos | Experiencia en administración de servidores, conocimiento de SQL, redes |
| Número aproximado de participantes | 2–4 administradores |
| Dedicación esperada | 4–5 horas por actuación + práctica avanzada |

## 6.3.3 Objetivos de formación

1. Administrar usuarios, permisos y roles en cada actuación.
2. Ejecutar procedimientos de backup, restauración y recuperación ante desastre.
3. Monitorizar salud del sistema y responder a alertas críticas.
4. Aplicar parches de seguridad y actualizaciones de forma segura.
5. Configurar integración con sistemas municipales existentes.
6. Auditar acceso y actividad para cumplimiento regulatorio.
7. Planificar capacidad y escalabilidad del sistema.

## 6.3.9 Evaluación de competencia final

Al completar la formación, cada administrador deberá ser capaz de:

- ✅ Crear usuario con permisos específicos y verificar acceso
- ✅ Ejecutar backup completo y restauración a punto específico
- ✅ Renovar certificado SSL sin downtime
- ✅ Publicar nueva versión de app con rollback automático si falla
- ✅ Actualizar firmware de equipamiento remoto
- ✅ Investigar anomalía de datos y determinar causa raíz
- ✅ Realizar auditoría de cumplimiento regulatorio (GDPR, etc.)
- ✅ Monitorizar sistema y responder a alertas críticas
- ✅ Documentar procedimiento post-intervención

**Se considerará completada cuando administrador complete todas las tareas autónomamente y de forma segura.**
