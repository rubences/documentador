# 2.13 Actuación 13: Modernización y Ampliación de la Red WiFi Municipal

## 2.13.1 Descripción general de la solución software

La actuación contempla el software de monitorización, analítica de uso y gestión lógica de la
red WiFi municipal modernizada, incluyendo telemetría de puntos de acceso, análisis anónimo
de hábitos de usuarios y propagación de una SSID privada para usos municipales.

Conforme al Pliego, el software asociado debe operar sobre la red renovada, incorporar
licencias de por vida, disponer de actualizaciones durante 10 años y servir tanto para el
mantenimiento de la red como para la monitorización turística del destino.
