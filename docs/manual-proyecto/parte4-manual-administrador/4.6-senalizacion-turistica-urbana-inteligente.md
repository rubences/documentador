# 4.6 Actuación 6: Señalización Turística Urbana Inteligente - Administración

## 4.6.1 Objeto del manual de administración

La presente sección describe las tareas de administración técnica del sistema de señalización
turística inteligente, incluyendo gestión de beacons, configuración de contenidos y control
de acceso.

## 4.6.2 Perfiles de administración

| Perfil | Responsabilidades |
| --- | --- |
| Administrador de beacons | Configuración de dispositivos BLE y gestión de contenidos |
| Responsable de QR/NFC | Creación de códigos dinámicos y mapeo a contenidos |
| Técnico de señalización | Mantenimiento físico y actualizaciones de hardware |

## 4.6.3 Acceso al panel de gestión de señalización

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder a la plataforma de administración de beacons. |
| 2 | Iniciar sesión con credenciales de administrador. |
| 3 | Verificar acceso a configuración de dispositivos y contenidos. |
| 4 | Confirmar disponibilidad de herramientas de mapeo geográfico. |

## 4.6.4 Gestión de beacons BLE

| Acción | Instrucción |
| --- | --- |
| Registrar nuevo beacon | Documentar UUID, major, minor y ubicación física. |
| Configurar parámetros de beacon | Ajustar potencia de transmisión y frecuencia de emisión. |
| Asignar contenido | Vincular beacon a mensaje o recurso que recibirán usuarios próximos. |
| Monitorizar batería | Revisar nivel de energía y planificar reemplazo. |

## 4.6.5 Gestión de códigos QR dinámicos

| Acción | Instrucción |
| --- | --- |
| Crear nuevo QR dinámico | Definir URL o contenido inicial. |
| Modificar destino de QR activo | Cambiar enlace sin necesidad de regenerar código impreso. |
| Generar código para impresión | Crear archivo para señal física o totem. |
| Auditar acceso a QR | Revisar estadísticas de escaneos y procedencia. |

## 4.6.6 Gestión de contenidos asociados

| Acción | Instrucción |
| --- | --- |
| Crear contenido para beacon | Redactar mensaje turístico contextual. |
| Asignar a múltiples señales | Distribuir mismo contenido a varios beacons o QR. |
| Programar activación temporal | Establecer fechas de validez para contenidos estacionales. |
| Validar visualización | Revisar cómo se muestra en dispositivos móviles reales. |

## 4.6.7 Seguridad y control de acceso

| Acción | Instrucción |
| --- | --- |
| Configurar autenticación de beacon | Habilitar verificación de identidad para beacon legítimos. |
| Gestionar permisos de edición | Asignar quién puede modificar contenidos de cada señal. |
| Auditar cambios | Registrar quién y cuándo modificó configuración de señales. |
| Detectar beacons falsos | Monitorizar señales no autorizadas en cobertura municipal. |

## 4.6.8 Mantenimiento y monitorización

| Acción | Instrucción |
| --- | --- |
| Revisar estado de beacons | Verificar disponibilidad y potencia de transmisión. |
| Actualizar firmware de beacon | Distribuir actualizaciones de software del dispositivo. |
| Reemplazar batería | Planificar y ejecutar sustitución de baterías agotadas. |
| Validar cobertura | Probar alcance real de cada beacon desde diferentes ubicaciones. |

## 4.6.9 Analítica y medición de impacto

| Acción | Instrucción |
| --- | --- |
| Consultar estadísticas de escaneos | Revisar número de interacciones por señal. |
| Analizar procedencia geográfica | Identificar origen de usuarios que interactúan. |
| Generar informe de alcance | Calcular impacto de campañas por ubicación. |
| Evaluar ROI de señales | Determinar efectividad comparativa de cada beacon/QR. |

## 4.6.10 Gestión de incidencias administrativas

| Incidencia | Actuación recomendada |
| --- | --- |
| Beacon no transmite | Revisar configuración, batería y potencia de transmisión. |
| QR no resuelve correctamente | Verificar URL/contenido vinculado y validar integridad de código. |
| Notificación no llega a usuario | Comprobar rangos de radio y compatibilidad del terminal móvil. |
| Contenido obsoleto | Actualizar vinculación y validar cambios en dispositivos reales. |

## 4.6.11 Buenas prácticas administrativas

1. Realizar auditoría mensual de beacons y códigos QR operativos.
2. Documentar cambios en mapeo de contenidos turísticos.
3. Probar periódicamente escaneo real en campo por múltiples dispositivos.
4. Mantener protocolo de reemplazo de baterías según vida útil estimada.
5. Revisar estadísticas para optimizar ubicación y mensaje de señales.
