# 3.8 Actuación 8: Sistema de Sensorización Ambiental

## 3.8.1 Objeto del manual de usuario

La presente sección describe el uso funcional del sistema de monitorización ambiental para
usuarios municipales encargados de consulta de indicadores, seguimiento de alertas y análisis
de evolución de parámetros ambientales.

## 3.8.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Operador ambiental | Supervisión diaria de estaciones y medidas |
| Técnico municipal | Gestión de alertas e incidencias de calidad ambiental |
| Analista de destino | Explotación histórica y elaboración de informes |

## 3.8.3 Acceso al sistema de monitorización

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al panel de monitorización integrado en la Plataforma de Ciudad Inteligente. |
| 2 | Iniciar sesión con credenciales habilitadas. |
| 3 | Seleccionar zona, estación y parámetros a visualizar. |
| 4 | Confirmar que la telemetría se recibe correctamente en tiempo real. |

## 3.8.4 Consulta operativa de parámetros

| Acción | Instrucción |
| --- | --- |
| Visualizar mapa geolocalizado | Localizar estaciones y estado de funcionamiento. |
| Consultar último dato por sensor | Revisar valor actual de cada parámetro ambiental. |
| Revisar estado de equipo | Consultar actividad, avería y batería cuando aplique. |
| Cambiar periodo de consulta | Alternar entre tiempo real y vistas históricas. |

## 3.8.5 Gestión de alertas

| Tipo de alerta | Actuación de usuario |
| --- | --- |
| Umbral máximo o mínimo | Revisar valor, validar contexto y notificar a responsable técnico. |
| Fuera de rango operativo | Verificar estado de sensor y registrar incidencia. |
| Repetición de ocurrencias | Analizar patrón temporal y elevar seguimiento específico. |
| Avería detectada | Abrir incidencia de mantenimiento según procedimiento municipal. |

## 3.8.6 Informes y explotación de datos

| Acción | Instrucción |
| --- | --- |
| Generar informe por zona | Seleccionar ubicación y periodo de análisis. |
| Comparar periodos | Contrastar evolución temporal de indicadores clave. |
| Exportar resultados | Descargar información en formatos habilitados por el sistema. |
| Compartir resultados internos | Publicar informe en los canales municipales definidos. |

## 3.8.7 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| Una estación no reporta datos | Revisar conectividad y estado de la última transmisión. |
| Medida anómala puntual | Validar con histórico cercano y registrar observación. |
| No se visualiza un indicador | Comprobar filtros activos y permisos del perfil. |
| No se reciben alertas | Revisar política de alertado y canales de notificación configurados. |

## 3.8.8 Buenas prácticas

1. Revisar diariamente la telemetría de todos los puntos activos.
2. Mantener criterios homogéneos de umbral y periodo para comparativas.
3. Registrar actuaciones ante alertas para asegurar trazabilidad.
4. Validar calidad de datos antes de su difusión pública o interdepartamental.
