# 3.9 Actuación 9: Sistema de Información de Aparcamientos Públicos

## 3.9.1 Objeto del manual de usuario

La presente sección describe el uso funcional del sistema de información de aparcamientos
públicos para personal operador y usuarios responsables de validación de ocupación,
publicación de plazas libres y corrección de desajustes de conteo.

## 3.9.2 Perfiles de uso

| Perfil | Uso principal |
| --- | --- |
| Operador de aparcamiento | Supervisión de ocupación y corrección manual de incidencias |
| Gestor municipal de movilidad | Seguimiento de disponibilidad en tiempo real |
| Usuario técnico funcional | Verificación de integraciones con soportes de difusión |

## 3.9.3 Acceso al sistema de control

| Paso | Instrucción |
| --- | --- |
| 1 | Acceder al entorno web del sistema de control de aparcamientos. |
| 2 | Iniciar sesión con credenciales autorizadas. |
| 3 | Verificar estado de recepción de conteos en entradas y salidas. |
| 4 | Confirmar actualización de plazas libres en paneles y canales digitales. |

## 3.9.4 Operación diaria

| Acción | Instrucción |
| --- | --- |
| Revisar ocupación en tiempo real | Consultar plazas disponibles por aparcamiento. |
| Verificar publicación | Confirmar actualización en paneles, web y app si aplica. |
| Comprobar integridad del conteo | Revisar coherencia entre entradas, salidas y ocupación total. |
| Registrar incidencias | Anotar desvíos y actuaciones realizadas. |

## 3.9.5 Corrección manual de errores de ocupación

| Acción | Instrucción |
| --- | --- |
| Acceder a la página de corrección | Entrar con perfil autorizado de control de aparcamiento. |
| Validar autenticación segura | Completar identificación con mecanismo de firma digital cuando aplique. |
| Ajustar ocupación | Introducir corrección necesaria sobre el valor real observado. |
| Guardar trazabilidad | Confirmar registro de modificación para auditoría posterior. |

## 3.9.6 Gestión de incidencias de usuario

| Incidencia | Actuación recomendada |
| --- | --- |
| El número de plazas no coincide con la realidad | Aplicar corrección manual desde el módulo de control. |
| No se actualizan los paneles | Revisar estado de comunicación con el sistema central. |
| No permite acceso al módulo de corrección | Verificar credenciales y mecanismo de firma de origen. |
| Desfase recurrente de conteo | Registrar incidencia y comunicar al equipo técnico para revisión del origen. |

## 3.9.7 Buenas prácticas

1. Realizar revisión periódica de coherencia de ocupación por turnos.
2. Registrar todas las correcciones manuales para facilitar auditoría.
3. Evitar ajustes sin contraste previo con observación operativa real.
4. Comunicar rápidamente incidencias repetitivas de conteo para su resolución técnica.
